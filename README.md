Sau đó tạo một file ".env" copy toàn bộ file ".env.example" chỉnh sửa lại kết nối CSDL MySQL<br>
Tiếp tục 3 lệnh sau "php artisan key:generate" -> "php artisan cache:clear" -> "php artisan config:clear"<br>
